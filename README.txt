Урок 4. Обработка исключений

1. Доделать задания про интернет-магазин. Обработать исключения,
показывать "говорящие" сообщения о том, что пошло не так.

2. В класс покупателя добавить перечисление с гендерами,
добавить в объекты покупателей свойство «пол» со значением
созданного перечисления. Добавить геттеры, сеттеры.

3. Добавить перечисление с праздниками (нет праздника,
Новый Год, 8 марта, 23 февраля). В метод оформления заказа
добавить параметр - дата заказа (можно сразу использовать
значение из перечисления), при создании заказа проверять
значение перечисления и пол покупателя. Делать скидку
мужчинам - на 23 февраля, женщинам на 8 марта в размере 15%,
всем на Новый год в размере 20%. В заказе указывать общую
сумму со скидкой.

в readme либо в тексте к ПЗ
